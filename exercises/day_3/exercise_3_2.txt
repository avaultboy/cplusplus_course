День 3 Задание 2 Объединить два упорядоченных списка
Имеется односвязный список:
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(NULL) {}
  };
  
Требуется написать функцию объединения двух отсоритированных списков в новый:
    ListNode* mergeTwoLists(ListNode* l1, ListNode* l2);

Новый список должен быть составлен путем соединения элементов первых двух списков.
Пример:
l1: 1->2->4
l2: 1->3->4
merged list: 1->1->2->3->4->4
