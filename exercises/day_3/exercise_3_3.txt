День 3 Задание 3 Сложение чисел
Даны два не пустых односвязных списка (определение списка см. ниже) представляющих
собой целые положительные числа. Разряды числа хранятся в списке в обратном порядке
и каждый элемент списка хранит один разряд.
Реализуйте функцию сложения двух таких "чисел", результат должен быть представлен также односвязным списком.

Пример:
    num1: 2->4->3
    num2: 5->6->4
    //(2->4->3) + (5->6->4)
    result: 7->0->8  //342 + 465 = 807


//определение списка
struct ListNode {
   int val;
   ListNode *next;
   ListNode(int x) : val(x), next(NULL) {}
};
