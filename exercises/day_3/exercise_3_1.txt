День 3 Задание 1 Сложные структуры
Описать структуру 'Building' c полями:
'название',
'владелец',
'адрес',
'дата постройки',
'количество этажей',
'количество квартир'.

Структура должна содержать указатель на квартиры в здании (private: Apartment* apartments)
и поле с максмиальным количеством квартир (maxNofAparts), память для 'apartments' должна
выделяться автоматически при добавлении новой квартиры
(сразу на размер maxNofAparts, если добавляется первая новая квартира).

Собственно тип Apartment является структурой с полями:
'размер',
'количество комнат',
'этаж',
'стоимость'
'владелец'.

Поля 'адрес' и 'дата' в структуре 'здание' так же являются структурами:
адрес:
    'город',
    'улица',
    'номер здания'
дата:
    'месяц', (перечисляемый тип)
    'год'

Для структуры 'Building' реализовать методы:

void Building::addApartment(const Apartment& apart);
    выделяет память для максмиального количества квартир (если первая квартира),
    добавляет квартиру в здание

Apartment* Building::getApartment(unsigned int number) const;
    возвращает указатель на квартиру по номеру в доме

unsigned int Building::getNofApartments(void) const;
    возвращает количество квартир в доме

double Building::getBuildingPrice(void) const;
    возвращает общую стоимость всех квартир в здании

void Building::clearApartments(void);
    удаляет все квартиры (освобождает память)

Реализовать функцию void showBuilding(const Building& building);
который будет выводить в поток вывода информацию о доме и всех квартирах
Пример:
